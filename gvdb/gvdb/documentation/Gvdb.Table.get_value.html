gvdb_table_get_value

GVariant* gvdb_table_get_value (GvdbTable* table,
                                const gchar* key);

Looks up a value named key in @file.

If the value is not found then NULL is returned. Otherwise, a new #GVariant instance is returned. The #GVariant does not depend on the continued existence of @file.

You should call g_variant_unref() on the return result when you no longer require it.

table
key

a string

Returns

a #GVariant, or NULL
